# YuZhai_2
御宅工作V2.0
该项目主要使用了MVP架构+RXJava+网络请求库Retrofit进行开发，项目分包主要按照功能进行划分

使用了常用的开发库，图片加载Glide，事件总线EventBus，ORM数据库Litepal，推送服务MiPush等
